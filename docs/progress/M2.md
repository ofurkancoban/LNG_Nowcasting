## Known caveat, carried forward honestly

The reference table is a **6-vessel starter set**, not the full global LNG
carrier fleet (~700 vessels) flagged as an open question in
`docs/decisions/0001-architecture.md`. It proves the registry/capacity
lookup pipeline works end to end with real, verifiable vessel data, but must
be expanded significantly before this can support real-world nowcasting
coverage. Expansion is not part of M2's acceptance criteria and is left as
a follow-up.
