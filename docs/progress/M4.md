## Known caveat, carried forward honestly

The exact ALSI status-flag field name (`status`, values E/C/N) was inferred
from the GIE API manual's general description of data quality indicators
(section 3.1) rather than an explicit field name in the ALSI field table in
section 2.4 of the manual (which lists `gasDayStart`, `inventory`,
`sendOut`, `dtmi`, `dtrs`, `Info`, `{children}` but does not itself name the
status field for ALSI specifically, unlike the AGSI table which does). This
mirrors the same honesty standard used for AISStream's field names in M1:
flagged here rather than silently asserted as certain. Confirm against a
live ALSI response before relying on this field name in production.
